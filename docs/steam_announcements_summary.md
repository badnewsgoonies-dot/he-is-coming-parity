Source: artifacts/he_is_coming_research/pages/https_store_steampowered_com_app_2824490_He_is_Coming.txt

## Friend Battle / Kingmaker Update
- Adds real-time 1v1 battles within Kingmaker (previously asynchronous vs uploaded builds).
- Both players start from the same seed; choices during the run determine the outcome.
- Framed as a new confrontation style, emphasizing simultaneous decision pressure.

- Fixed MasterChef Steam achievement not triggering.

- Junksmith no longer allows duplicate of the same edge.
- Campfire fog-of-war lighting now limited to tiles explored by the player (smaller radius).
